How it works?

We are using gulp-file-include plugin which has it's own set of functions with which you can use partials.

// example of how buttons are rendered in this page
@@include ("./components/button.html")

// similarly the partials like
// head, header or footer are from partials folder
@@include ("./partials/header.html")

//note: remmove space after @@include for rendering html file

Read more about API and other options in package documentation

Note: You might have to disable formatting in Code Editor like VSCode if you are using dynamic options in @@include command are it's parsing will be affected like new line character etc.
